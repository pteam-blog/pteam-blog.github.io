pteam-blog
=====

Open source Blog built by jekyll.  
Demo: http://pteam-blog.github.io/

### Features
- Responsive layout.
- Supports tags and categories.
- Social profile and bio of author.
- Bootstrap based.
- Glyphicon and Font-Awesome Icons.
- Pagination.
- Syntax highlighting with pygments.
- Disqus comments.
